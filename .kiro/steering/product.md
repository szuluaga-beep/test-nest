# Product Overview

## Summary
A NestJS-based REST API application with PostgreSQL database integration. The project provides a scalable backend foundation for managing users and events with TypeORM for data persistence and full migration support.

## Core Modules
- **Users**: User management endpoints and database operations
- **Events**: Event management endpoints and database operations
- **App**: Core application setup and health check endpoints

## Key Features
- RESTful API endpoints for users and events
- PostgreSQL database with TypeORM ORM
- Database migrations with version control
- Docker containerization support
- Comprehensive test coverage (unit, integration, and e2e tests)
- Swagger API documentation support
- Input validation using class-validator
- Environment configuration management

## Technology Stack
- **Framework**: NestJS 11.x
- **Language**: TypeScript 5.x
- **Database**: PostgreSQL with TypeORM 1.x
- **Package Manager**: pnpm
- **Testing**: Jest with supertest for e2e
- **Validation**: class-validator, class-transformer
- **Code Quality**: ESLint, Prettier
